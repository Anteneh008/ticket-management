# Tech Stack

## Frontend
- **Next.js** (App Router) + **TypeScript** — full-stack framework handling both UI and API routes
- **Tailwind CSS** + **shadcn/ui** — accessible UI components

## Database
- **PostgreSQL** — relational structure for tickets, users, categories, and statuses
- **Prisma** — type-safe ORM

## Authentication
- **NextAuth.js** — session management and role-based access (admin vs. agent)

## AI
- **Anthropic Claude API** — ticket classification, summaries, and suggested replies

## Email
- **Hostinger Mail** — sending replies and receiving inbound support emails

## Deployment
- **Vercel** — zero-config deployment for Next.js
